# Projekt: VinLogistics (Arbeitstitel)

## Ziel
Eine hochperformante iOS-App zur Verwaltung von Weinkellern mit Fokus auf Logistik, Trinkreife und Offline-Fähigkeit.

## Kernfunktionen
1. Zwei Lagerorte (Keller vs. Wohnung) + Verwaltung von Subskriptionen (Zulauf).
2. On-Device OCR-Scan für Einlagerung, Umlagerung und Austragung.
3. Inventur-Modus mit Soll/Ist-Abgleich.
4. Trinkreifen-Analyse basierend auf konfigurierbaren Zeitfenstern.

## Technischer Stack
- Sprache: Swift (SwiftUI)
- Datenbank: SwiftData (lokal)
- OCR: Vision Framework
- Architektur: Modulare Clean Architecture
